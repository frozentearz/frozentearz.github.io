子查询

通过把一个查询的结果作为另一个查询的一部分,子查询一般出现在 SELECT 语句的 WHERE 子句中，Oracle 也支持在 FROM 或 HAVING 子句中出现子查询。

子查询比主查询先执行，结果作为主查询的条件，在书写上要用圆括号扩起来，并放在比较运算符的右侧。
